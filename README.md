# Crime Map
범죄데이터 시각화

## Stack
- Node.js(express)
- React.js
- MongoDB
- state : Mobx
- Webpack
- Typescript
- GraphQL
- [SD2 Admin Bootstrap 템플릿](https://startbootstrap.com/themes/sb-admin-2/)

## GIS데이터

- 포맷 : WKT
- 에디터 : QGIS [참고문서](https://dominoc925.blogspot.com/2018/01/using-qgis-to-export-csv-files-with.html)
- 주의사항 : 좌표계를 EPSG:4326 - WGS 84로 해야함
- 시도 : 2015년
- 시군구 : 2016년
